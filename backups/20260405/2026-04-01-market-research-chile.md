## 3️⃣ DISEÑO DEL MVP DE CHATBOT WHATSAPP

### Alcance del MVP (Mínimo Producto Viable)

**Objetivo:** Probar que un corredor puede recibir leads, calificarlos automáticamente, y agendar visitas sin intervención manual.

**Scope del MVP (2-3 semanas de desarrollo):**

```
┌─────────────────────────────────────────────────────────────┐
│  LEAD ENTRA POR:                                            │
│  • WhatsApp Business API                                    │
│  • Formulario web (embed)                                   │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  CHATBOT (IA)                                               │
│  • Saluda y presenta al corredor                            │
│  • Pregunta: ¿compra, vende o arrienda?                     │
│  • Pregunta: ¿zona de interés?                              │
│  • Pregunta: ¿presupuesto aproximado?                       │
│  • Pregunta: ¿urgencia? (esta semana, este mes, + tarde)    │
│  • Pregunta: ¿horario preferido para visita?                │
└─────────────────────────────────────────────────────────────┘
                            ↓
┌─────────────────────────────────────────────────────────────┐
│  DECISIÓN                                                   │
│  SI califica (presupuesto + urgencia):                     │
│    → Ofrece 2-3 opciones de horario                         │
│    → Agenda en Google Calendar del corredor                 │
│    → Envía confirmación con dirección de la propiedad       │
│                                                             │
│  SI NO califica (sólo curioso):                            │
│    → Guarda en lista de seguimiento                         │
│    → Recontacta en 7 días con propiedades similares         │
└─────────────────────────────────────────────────────────────┘
```
