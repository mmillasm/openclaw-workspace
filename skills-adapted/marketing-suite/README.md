## 🎯 Cómo Usar

### Skill: Competitor Analysis

**Cuándo usar:**
- Antes de una llamada de ventas
- Investigando un competidor específico
- Preparando una propuesta diferenciada

**Uso:**
```
Analiza competidor: https://godevs.cl
```

**Output:** `competitor-analysis-godevs.md`

---

### Skill: Website Audit

**Cuándo usar:**
- Investigando una agencia prospecto
- Preparando un pitch personalizado
- Identificando pain points específicos

**Uso:**
```
Audita website: https://gescor.cl
```

**Output:** `website-audit-gescor.md`

---

### Template: Secuencia de Bienvenida

**Uso:** En Mailchimp/Beehiiv cuando alguien descarga un lead magnet

**Emails incluidos:**
1. Bienvenida + Entrega (inmediato)
2. El Problema Real (día 2)
3. Prueba Social (día 4)
4. Superar Objeción (día 6)
5. La Pregunta (día 8)

**Personalizar:**
- [Nombre del lead]
- Casos de estudio (Pedro, María)
- Links a demos

---

### Template: Propuesta de Servicio

**Uso:** Enviar a agencias interesadas después de una llamada de descubrimiento

**Secciones:**
- Resumen Ejecutivo
- Análisis de Situación
- Planes y Precios (Starter/Pro/Agency)
- Proyección ROI
- Implementación
- Casos de éxito
- Garantía

**Personalizar:**
- Datos específicos de la agencia
- Métricas de impacto estimadas
- Casos de estudio relevantes

## 🛠️ Adaptaciones Realizadas

### Del Repo Original a VYNXE

**Contexto:**
- De genérico/marketing → Especializado/inmobiliario
- De inglés → Español (Chile)
- De SaaS general → Automatización WhatsApp para corredores

**Tonos:**
- De corporativo → Directo y amigable
- De técnico → Sin jerga (como pide Ader)
- De ventas agresivo → Consultivo y útil

**Ejemplos:**
- "Lead scoring" → "Detectamos quién quiere comprar"
- "API integration" → "Se conecta con tu calendario"
- "Conversion optimization" → "Más citas agendadas"
