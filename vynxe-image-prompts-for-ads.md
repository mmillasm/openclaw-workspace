## Guía Rápida de Uso

### Herramientas recomendadas para generar:

1. **Leonardo.ai** — Para imágenes de estilo real/fotográfico
2. **Ideogram** — Para texto en imágenes (maneja español mejor)
3. **Midjourney** — Para estilo artístico
4. **Canva** — Para carousels y composiciones finales

### Flujo de trabajo:

```
1. Generar imágenes base con prompts arriba
2. Importar a Canva
3. Agregar logos VYNXE
4. Agregar texto adicional si necesario
5. Exportar en 1080x1080 (feed) o 1080x1920 (reels/stories)
```

---

## Especificaciones Técnicas

| Formato | Dimensiones | Ratio |
|---------|-------------|-------|
| Feed | 1080 x 1080 | 1:1 |
| Reels/Stories | 1080 x 1920 | 9:16 |
| Landscape | 1920 x 1080 | 16:9 |
